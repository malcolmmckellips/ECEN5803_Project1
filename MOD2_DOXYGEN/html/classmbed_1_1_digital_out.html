Detailed Description

A digital output, used for setting the state of a pin

Note
Synchronization level: Interrupt safe

Example:

// Toggle a LED
#include "mbed.h"
DigitalOut led(LED1);
int main() {
while(1) {
led = !led;
wait(0.2);
}
}
